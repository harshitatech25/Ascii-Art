# Ascii-Art(Python)
ASCII art text generator built in Python with dynamic styling and terminal color formatting.
A Python-based ASCII art and text-styling tool that prints stylized characters, colors, and terminal effects using simple functions. This project demonstrates formatted output, dynamic styling, and interactive terminal text display.

**Features**
ASCII-style text output
Colored text using colorama
Dynamic terminal effects
Easy to customize
Beginner-friendly Python code

**Requirements**
Install the required package:
pip install colorama
How to Run
Run the Python script:
python main.py
